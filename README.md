# MERN Student Management System

A full-stack **Student Management System** built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js).  
This application allows CRUD operations (Create, Read, Update, Delete) on student records.

---

## Features

- Add new students with details like name, email, course, and phone number
- View all students in a table format
- Update student information
- Delete student records
- Responsive UI built with React
- RESTful API backend with Node.js and Express
- MongoDB database for data storage

---

## Tech Stack

| Layer         | Technology           |
|---------------|--------------------|
| Frontend      | React.js, HTML, CSS |
| Backend       | Node.js, Express.js |
| Database      | MongoDB, Mongoose   |
| Version Control | Git & GitHub       |
